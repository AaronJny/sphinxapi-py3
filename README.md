sphinxapi-py3
=============

Python 3 port of official python client for Sphinx Search http://sphinxsearch.com/

### Installation

```sh
pip install https://github.com/a1tus/sphinxapi-py3/zipball/master
```

### Using

```sh
import sphinxapi
```
